# MorningsideZW — Static site (Tailwind CDN) + Decap CMS

This archive contains a ready-to-deploy static site for `morningsidezw.com`.

## Quick deploy
1. Unzip and push the archive contents to a GitHub repo named `morningsidezw` (public).
2. Add `CNAME` with `morningsidezw.com` (included as file below) or set custom domain in Pages settings.
3. In GitHub repository settings → Pages, enable Pages from `main` branch / root.
4. Visit https://morningsidezw.com/admin to access Decap CMS (after GitHub OAuth or Netlify Identity setup).

## Notes
- This version uses Tailwind CDN for zero-build deployment.
- To use a compiled Tailwind workflow, replace the CDN and run the build scripts included in the full project repo.

This ZIP contains everything needed to run the site on GitHub Pages.
